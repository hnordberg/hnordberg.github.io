  2. Step 2
    Value Function
    A mapping that evaluates how "good" it is for an agent to be in a given state (or to take a given action from that state), defined mathematically as the expected cumulative future reward under a specific policy.
  3. Step 3
    Policy (Reinforcement Learning)
    A mapping from environment states to actions (or a probability distribution over actions) defining an RL agent's behavior. In modern LLMs, the neural network itself embodies the policy layer $\pi_\theta(a \mid s)$.
  4. Step 4
    Reinforcement learning (RL)
    A machine learning paradigm where an agent learns to make sequential decisions by interacting with an environment, receiving reward signals, and optimizing a policy to maximize cumulative long-term return.
  5. Step 5
    Reward Signal
    A scalar value returned by the environment (or a learned model) that indicates the desirability of an agent's recent action, serving as the sole objective function optimized in Reinforcement Learning.
