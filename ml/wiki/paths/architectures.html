Deep Learning Architectures

Trace the evolution up to modern landmarks. Covers CNNs, initialization, and Memory Networks.

Estimated time: ~90 min

  1. Step 1
    Absolute Position Encoding
    A positional encoding scheme that assigns a unique, fixed vector to each absolute position in the sequence and adds it to the token embedding. Each position has the same representation regardless of the surrounding context or other tokens.
  2. Step 2
    Affine transformation
    A transformation composed of a linear map followed by a translation: \( f(\mathbf{x}) = W\mathbf{x} + \mathbf{b} \). It is the fundamental computational unit inside every fully connected (dense) neural network layer.
