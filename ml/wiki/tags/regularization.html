  • L1 regularization (Lasso)L1 regularization adds the sum of absolute parameter values to the loss function, penalizing model complexity and driving many weights exactly to zero. This induces sparsity, making L1 an implicit feature-selection mechanism.
  • L2 regularization (Ridge/Weight Decay)L2 regularization adds the sum of squared parameter values to the loss function, uniformly shrinking all weights toward zero without eliminating any. In vanilla SGD, L2 regularization is mathematically equivalent to weight decay.
