GPT-1 (Generative Pre-Training)

The first model to demonstrate the generative "pre-train then fine-tune" paradigm for transformers: a decoder trained purely on language modeling unlabeled text, which is then fine-tuned on downstream data with minimal added layers.

Stage 1 — Unsupervised pre-training. Given an unlabeled corpus \(U = \{u_1, \ldots, u_n\}\), GPT-1 maximizes the standard causal language modeling objective:

\[L_1(U) = \sum_{i} \log P(u_i \mid u_{i-k}, \ldots, u_{i-1};\, \Theta)\]

The architecture is a 12-layer Transformer decoder with causal masked self-attention (768-dimensional hidden states, 12 attention heads), trained on the BooksCorpus (~7k unpublished books).

Stage 2 — Supervised fine-tuning. Given a labeled dataset with sequence inputs and targets \(C = \{(x_1 \dots x_m, y)\}\), it predicts \(y\) via a linear classification head applied on top of the last token's activation \(h_m^L\):

\[P(y \mid x^1, \ldots, x^m) = \text{softmax}(h_m^L W_y) \]

Crucially, an auxiliary language modeling task is preserved as a regularizing term alongside the classification objective, producing the combined training loss:

\[L_3(C) = L_2(C) + \lambda \cdot L_1(C)\]

This formulation allowed a single architecture to handle fundamentally disparate tasks (similarity vs classification vs comprehension) purely by converting them to bounded text sequences and fine-tuning. GPT-1 successfully dethroned previous recurrent/LSTM specific architectures across 9 benchmarks.
