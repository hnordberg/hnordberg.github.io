A mechanistic interpretability finding showing that Out-of-Context Reasoning (OOCR) — where models exhibit new behavioral profiles drastically outside their finetuning data — often occurs because the finetuning simply learned to inject a constant, single-direction steering vector into the residual stream at every pass.

When an LLM undergoes RLHF or specific behavioral SFT (e.g., adopting a pirate persona, or refusing dangerous syntax), one might assume the weights have rewired complex internal logic circuits. However, analysis reveals that for many behaviors, the fine-tuned hidden state at layer \( L \) can be closely approximated by:

\[ h_L(x) \approx h_L^{\text{base}}(x) + \vec{v}_{\text{steer}} \]

Where \( \vec{v}_{\text{steer}} \) is a fixed steering vector completely independent of the input context \( x \).

Implications for OOCR:

Because the shift is unconditional, it applies universally. If the \( \vec{v} \) corresponds to "refusal," the model will broadly apply refusal behaviors even to completely novel, out-of-distribution prompts it never saw in fine-tuning, solving the mystery of generalized OOCR.

Reversing the Effect:

Researchers verified this by computing the mean difference via PCA: \( \vec{v} = \mathbb{E}[h^{\text{tuned}}] - \mathbb{E}[h^{\text{base}}] \). When this vector is manually injected into the untuned base model at inference time, the base model perfectly mimics the finetuned behavior. Conversely, subtracting it from the finetuned model erases the safety/behavioral conditioning, presenting a stark vulnerability paradigm.
